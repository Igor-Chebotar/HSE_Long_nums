Библиотека определяет класс «Длинное Число», представляющий число с фиксированной точкой заданной точности в двоичной системе счисления.

Определение класса и объявление его методов находятся в заголовочном файле include/LongNumber.hpp

Реализация класса находится в src/LongNumber.cpp

Демонстрация работы библиотеки представлена в src/main.cpp

Реализация расчёта числа pi представлена в файле src/pi_calculator.cpp(Корректно выполняется с точностью до 13 знаков после запятой)

Сборка осуществляется при помощи системы сборки Make:

- Команда сборки проекта: make

- Команда запуска тестов: make test

- Команда запуска вычисления 100 знаков числа Пи: make pi
